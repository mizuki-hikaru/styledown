## About This Example

This folder is meant to be converted into a small static site.

- `about.md` becomes `about.html`
- `blog/post.md` becomes `blog/post.html`
- The converter will create `index.html` for directories that don't have `index.md`

Try converting the `example/` directory and then click around:

- the breadcrumb at the top should update as you move between pages
- directories without `index.md` should show a directory listing
