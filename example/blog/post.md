# First Post

This is a tiny page to demonstrate breadcrumbs for nested paths.

- The breadcrumb should show: Home / Blog / Post
- The directory `blog/` intentionally has no `index.md`, so it should get a generated `blog/index.html` listing.

Go back to the blog listing by clicking the breadcrumb link for `Blog`.
